# Drupal CMS Forms

Adds a simple contact form, and tools for building forms.

## Recipes Applied

- [Drupal CMS Anti-spam](https://www.drupal.org/project/drupal_cms_anti_spam)

## Modules and Themes Installed

- [CAPTCHA](https://www.drupal.org/project/captcha)
- Editor (core)
- Filter (core)
- Views (core)
- [Webform](https://www.drupal.org/project/webform)

## Installation

This project is part of [Drupal CMS](https://drupal.org/project/cms). If you're
not using Drupal CMS, you can still use this recipe in your own project. Add it
with the following Composer command:

```
composer require drupal/drupal_cms_forms
```
