# Drupal CMS Authentication Tweaks

Adds useful authentication features.

## Modules and Themes Installed

- [Drupal CMS Helper](https://www.drupal.org/project/drupal_cms_helper)
- [ECA](https://www.drupal.org/project/eca)
- [Login Email or Username](https://www.drupal.org/project/login_emailusername)
- [Token](https://www.drupal.org/project/token)

## Installation

This project is part of [Drupal CMS](https://drupal.org/project/cms). If you're
not using Drupal CMS, you can still use this recipe in your own project. Add it
with the following Composer command:

```
composer require drupal/drupal_cms_authentication
```
